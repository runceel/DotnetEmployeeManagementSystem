## ユーザーインタラクション

### ログインフロー

1. ユーザーが `/login` にアクセス
2. テストユーザーチップをクリック（オプション）
3. ユーザー名とパスワードが自動入力
4. ログインボタンをクリック
5. ローディングスピナー表示
6. 成功時:
   - 成功通知（Snackbar）
   - ホーム画面にリダイレクト
   - 認証バナー表示
7. 失敗時:
   - エラーメッセージ表示
   - エラー通知（Snackbar）
   - 入力フィールドをクリア

### ログアウトフロー

1. ナビゲーションメニューまたはホーム画面から「ログアウト」をクリック
2. 認証情報クリア
3. ホーム画面にリダイレクト
4. 情報バナー表示
