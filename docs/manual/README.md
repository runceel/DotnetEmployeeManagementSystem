# ユーザーマニュアル

従業員管理システムのエンドユーザー向けマニュアルです。

## 📖 マニュアル構成

このマニュアルは、システムを実際に使用するエンドユーザー（従業員や管理者）向けに書かれています。技術的な詳細よりも、日常的な操作方法を中心に説明しています。

### 章立て

- **[第2章: ログイン・ログアウト](02-login.md)** - システムへのログイン・ログアウト手順
  - 初回アクセス
  - ログイン方法（通常ユーザー・テストユーザー）
  - ログアウト方法
  - エラー対応
  - セキュリティのベストプラクティス

### 今後追加予定の章

- **第1章: システム概要** - システムの目的と基本的な使い方
- **第3章: ダッシュボード** - ダッシュボード画面の見方と使い方
- **第4章: 従業員管理** - 従業員情報の閲覧・登録・編集
- **第5章: 勤怠管理** - 勤怠情報の入力と確認
- **第6章: 通知機能** - システム通知の確認と設定

## 🎯 対象読者

### エンドユーザー
- 従業員管理システムを日常的に使用する方
- 管理者権限を持つ方
- システムの基本操作を学びたい方

### このマニュアルに含まれないこと

以下の内容は、別のドキュメントを参照してください：

- **技術的な実装詳細** → [開発ガイド](../development-guide.md)
- **システムのセットアップ** → [Getting Started](../getting-started.md)
- **アーキテクチャ設計** → [アーキテクチャ概要](../architecture.md)
- **API仕様** → 各サービスのAPIリファレンス

## 📝 マニュアルの読み方

### 初めての方

1. [第2章: ログイン・ログアウト](02-login.md)から始めてください
2. ログインできたら、次の章（追加予定）で機能を学んでいきます

### 特定の機能を知りたい方

目次から該当する章を直接参照してください。各章は独立して読めるように構成されています。

## 💡 ヒント

- **実際に操作しながら読む**: 画面を見ながらマニュアルを読むと理解しやすいです
- **テストユーザーで練習**: 本番データを触る前に、テストユーザーで操作を練習できます
- **FAQ を確認**: よくある質問は各章のFAQセクションに記載されています

## 🆘 サポート

マニュアルに記載されていない問題や質問がある場合：

1. 各章の「よくある質問（FAQ）」セクションを確認
2. 「トラブルシューティング」セクションを確認
3. システム管理者に連絡
4. GitHubのIssueで報告（技術的な問題の場合）
