## 🎯 対象読者

### 一般ユーザー（User ロール）
- 従業員情報の閲覧
- 自分の勤怠記録の管理
- 休暇申請
- 通知の確認

### 管理者（Admin ロール）
- 上記すべての機能
- 従業員の登録・編集・削除
- 休暇申請の承認・却下
- 手動通知の送信
- システム管理

## 📖 使い方

1. 目次から必要な章を選択してください
2. 各章は独立して読むことができます
3. 画面の操作方法は、実際のシステムを操作しながら読むことをお勧めします
