# ユーザーマニュアル

従業員管理システムのエンドユーザー向けマニュアルです。

## 📖 マニュアル構成

このマニュアルは、システムを実際に使用するエンドユーザー（従業員や管理者）向けに書かれています。技術的な詳細よりも、日常的な操作方法を中心に説明しています。

### 章立て

- **[第1章: はじめに・システム概要](01-intro.md)** - システムの目的と利用方法
  - システムの目的と概要
  - サポート対象の利用環境・ブラウザ
  - 利用の前提（アカウント発行、パスワードなど）

- **[第2章: ログイン・ログアウト](02-login.md)** - システムへのログイン・ログアウト手順
  - 初回アクセス
  - ログイン方法（通常ユーザー・テストユーザー）
  - ログアウト方法
  - エラー対応
  - セキュリティのベストプラクティス

- **[第3章: ホーム画面](03-home.md)** - ホーム画面の見方と基本操作
  - ホーム画面の構成
  - ナビゲーションメニュー
  - ユーザー情報の表示

- **[第4章: ダッシュボード](04-dashboard.md)** - ダッシュボード画面の使い方
  - システム全体の概要把握
  - 統計カードの見方
  - 最近の活動タイムライン
  - クイックアクション
  - システムステータス

- **[第5章: 従業員管理](05-employees.md)** - 従業員情報の閲覧・登録・編集
  - 従業員一覧の表示
  - 従業員情報の検索・フィルタリング
  - 従業員の新規登録
  - 従業員情報の編集・削除

- **[第6章: 勤怠管理](06-attendance.md)** - 勤怠記録の確認、検索、集計方法
  - 勤怠履歴の検索・表示
  - 期間指定・従業員指定方法
  - 月次集計の使い方
  - 活用シーン別の使い方ガイド

### 今後追加予定の章

- **第7章: 通知機能** - システム通知の確認と設定

## 📝 マニュアルの読み方

### 初めての方

1. [第1章: はじめに・システム概要](01-intro.md)でシステム全体を理解
2. [第2章: ログイン・ログアウト](02-login.md)でログイン方法を学習
3. [第4章: ダッシュボード](04-dashboard.md)でダッシュボードの見方を確認
4. 必要に応じて各機能の章を参照

### 特定の機能を知りたい方

目次から該当する章を直接参照してください。各章は独立して読めるように構成されています。

## 💡 ヒント

- **実際に操作しながら読む**: 画面を見ながらマニュアルを読むと理解しやすいです
- **テストユーザーで練習**: 本番データを触る前に、テストユーザーで操作を練習できます
- **FAQ を確認**: よくある質問は各章のFAQセクションに記載されています

## 🆘 サポート

マニュアルに記載されていない問題や質問がある場合：

1. 各章の「よくある質問（FAQ）」セクションを確認
2. 「トラブルシューティング」セクションを確認
3. システム管理者に連絡
4. GitHubのIssueで報告（技術的な問題の場合）
