# ユーザーマニュアル

従業員管理システムのエンドユーザー向けマニュアルです。

## 📖 マニュアル構成

本マニュアルは、システムを利用する一般従業員および管理者向けに作成されています。

### 現在利用可能な章

#### [第11章: 付録](11-appendix.md)
- **11.1 主要用語解説** - システムで使用される用語の説明
- **11.2 権限ロール一覧** - AdminとUserロールの詳細と機能マトリックス
- **11.3 通知タイプ一覧** - システムが送信する通知の種類と内容
- **11.4 操作範囲・制限事項** - データ制限、技術的制限、セキュリティ制限
- **11.5 問い合わせ窓口・連絡先** - サポート窓口とFAQ

### 計画中の章（予定）

以下の章は今後追加される予定です：

- **第1章: はじめに** - システム概要、本マニュアルの使い方
- **第2章: ログインとアカウント管理** - ログイン方法、パスワード管理
- **第3章: 従業員情報管理** - 従業員一覧の閲覧、詳細確認
- **第4章: 勤怠管理** - 出退勤記録、勤怠履歴、月次集計
- **第5章: 休暇管理** - 休暇申請、承認フロー
- **第6章: 通知管理** - 通知の確認方法、手動通知の送信
- **第7章: 管理者機能** - 従業員の登録・編集・削除
- **第8章: レポートと集計** - 各種レポートの閲覧と出力
- **第9章: トラブルシューティング** - よくあるエラーと対処法
- **第10章: セキュリティとベストプラクティス** - パスワード管理、アクセス管理

## 🎯 対象読者

### 一般ユーザー（User ロール）
- 従業員情報の閲覧
- 自分の勤怠記録の管理
- 休暇申請
- 通知の確認

### 管理者（Admin ロール）
- 上記すべての機能
- 従業員の登録・編集・削除
- 休暇申請の承認・却下
- 手動通知の送信
- システム管理
