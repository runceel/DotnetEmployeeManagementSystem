# 第3章: ホーム画面の見方

## 3.1 概要

ホーム画面は、従業員管理システムにログインした後に最初に表示される画面です。システム全体の入口として、各機能へのアクセスポイントとなり、現在のログイン状態やシステムの主要機能を一目で確認できます。

## 3.2 画面構成

ホーム画面は以下の要素で構成されています：

### 3.2.1 画面上部（アプリケーションバー）

画面最上部には、システム全体で共通のアプリケーションバーが表示されます。

- **メニューボタン（左端）**: ハンバーガーメニューアイコン（☰）をクリックすると、左側のナビゲーションメニューの表示/非表示を切り替えられます
- **システム名**: 中央に「従業員管理システム」と表示されます
- **その他メニュー（右端）**: 縦に3つ並んだドットアイコン（⋮）が表示されます

### 3.2.2 ナビゲーションメニュー（左側サイドバー）

画面左側には、システムの主要機能にアクセスするためのナビゲーションメニューが配置されています。

#### 主要機能メニュー

| メニュー項目 | アイコン | 説明 |
|------------|---------|------|
| ホーム | 🏠 | 現在のページ。システムの概要を確認できます |
| ダッシュボード | 📊 | システムの統計情報や重要な概要を表示します |
| 従業員管理 | 👥 | 従業員情報の登録・編集・検索を行います |
| 勤怠管理 | 🕐 | 従業員の勤怠情報を管理します |
| 通知管理 | 🔔 | システムからの通知を確認します |

#### 認証関連メニュー

ナビゲーションメニューの下部には、ログイン状態に応じて以下が表示されます：

**ログイン時:**
- 現在のユーザー名が表示されます（人物アイコン付き）
- 「ログアウト」リンク

**未ログイン時:**
- 「ログイン」リンク

### 3.2.3 認証情報の表示

ホーム画面の上部には、現在のログイン状態を示すアラートメッセージが表示されます。

#### ログイン中の場合

緑色の成功メッセージが表示され、以下の情報が確認できます：

```
✓ ログイン中: ユーザー名 (メールアドレス)
```

**表示例:**
```
✓ ログイン中: admin (admin@example.com)
```

この表示により、現在どのユーザーでログインしているかが明確に分かります。

#### 未ログインの場合

青色の情報メッセージが表示されます：

```
ℹ ログインしていません。ログインページからログインしてください。
```

「ログインページ」の部分はリンクになっており、クリックするとログイン画面に遷移します。

### 3.2.4 ウェルカムメッセージ

画面中央上部に「従業員管理システムへようこそ」という見出しが表示され、その下に簡単なシステムの説明文が記載されています。

### 3.2.5 機能カード

ホーム画面中央には、3つの主要機能を紹介するカードが並んで表示されます。

#### ダッシュボードカード

- **アイコン**: 📊 ダッシュボード
- **説明**: システムの概要と重要な統計情報を確認できます
- **アクション**: 「開く」ボタンをクリックすると、ダッシュボード画面に遷移します

#### 従業員管理カード

- **アイコン**: 👥 従業員管理
- **説明**: 従業員情報の登録、編集、検索を行えます
- **アクション**: 「開く」ボタンをクリックすると、従業員管理画面に遷移します

#### 認証カード

- **アイコン**: 🔒 認証
- **説明**: ユーザー認証とアクセス管理を行います
- **アクション**: 
  - ログイン済みの場合: 「ログアウト」ボタンが表示されます
  - 未ログインの場合: 「ログイン」ボタンが表示されます

### 3.2.6 機能概要リスト

画面下部には、システムの技術的な特徴が箇条書きで表示されています：

- ✓ MudBlazorによるモダンなUI
- ✓ .NET Aspireによるマイクロサービス統合
- ✓ 従業員サービスとの連携（実装済み）
- ✓ 認証サービスとの連携（実装済み）

## 3.3 ナビゲーションの使い方

### 3.3.1 機能画面への移動

画面上部のメニューから各機能にアクセスできます。主な移動方法は以下の通りです：

1. **左側のナビゲーションメニューから選択**
   - メニュー項目をクリックすると、該当する機能画面に遷移します
   - 現在表示中のページは、ナビゲーションメニュー内でハイライト表示されます

2. **ホーム画面の機能カードから移動**
   - 各カードの「開く」ボタンをクリックすることで、直接該当機能にアクセスできます

3. **アプリケーションバーのホームアイコンから戻る**
   - どの画面からでも、ナビゲーションメニューの「ホーム」をクリックすると、このホーム画面に戻ることができます

### 3.3.2 ナビゲーションメニューの表示/非表示

- 画面左上のハンバーガーメニューアイコン（☰）をクリックすることで、ナビゲーションメニューを開閉できます
- 画面領域を広く使いたい場合は、メニューを閉じて作業することも可能です
- メニューを閉じても、アイコンをクリックすればいつでも再表示できます

## 3.4 認証情報の確認方法

現在ログインしているユーザーの情報は、以下の2箇所で確認できます：

1. **ホーム画面上部のアラートメッセージ**
   - ユーザー名とメールアドレスが表示されます
   - 最も分かりやすく、情報が豊富です

2. **左側ナビゲーションメニュー下部**
   - ログイン中のユーザー名が表示されます
   - どの画面からでも確認できます

これらの表示により、誤ったアカウントでログインしていないか、いつでも確認することができます。

## 3.5 ログイン・ログアウト

### 3.5.1 ログイン

ログインしていない状態でシステムを利用している場合：

1. ホーム画面上部の青いアラートメッセージ内の「ログインページ」リンクをクリック
2. または、左側ナビゲーションメニュー下部の「ログイン」をクリック
3. または、ホーム画面の認証カードの「ログイン」ボタンをクリック

これらのいずれかの方法でログイン画面に遷移し、認証情報を入力できます。

### 3.5.2 ログアウト

ログアウトする場合は、以下のいずれかの方法を使用します：

1. 左側ナビゲーションメニュー下部の「ログアウト」をクリック
2. ホーム画面の認証カードの「ログアウト」ボタンをクリック

ログアウトすると、ホーム画面に戻り、未ログイン状態になります。

## 3.6 画面の特徴

### 3.6.1 レスポンシブデザイン

ホーム画面は、さまざまな画面サイズに対応したレスポンシブデザインを採用しています：

- **大画面（デスクトップ）**: 機能カードが3列に並んで表示されます
- **中画面（タブレット）**: 機能カードが2列に並んで表示されます
- **小画面（スマートフォン）**: 機能カードが1列に縦に並んで表示されます

### 3.6.2 リアルタイム更新

ログイン状態の変更は、画面をリロードせずにリアルタイムで反映されます。ログイン・ログアウト操作を行うと、即座に表示内容が更新されます。

### 3.6.3 視覚的なフィードバック

- ログイン中は緑色の成功メッセージ（✓付き）
- 未ログインは青色の情報メッセージ（ℹ付き）
- 機能ごとに分かりやすいアイコンを使用

これらの視覚的な工夫により、現在の状態や各機能の役割を直感的に理解できます。
